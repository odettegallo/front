Preguntas y respuestas

Preguntas del Curso

  • ¿Qué es el desarrollo web?
    El desarrollo web es el proceso de creación, construcción y mantenimiento de sitios web o aplicaciones web. Involucra diversas tecnologías y disciplinas, desde la programación y el diseño hasta la gestión de bases de datos y la seguridad en línea.
  • Diferencias entre Front-End, Back-End y Fullstack.
    Existen tres principales tipos de desarrollo web:

    • Desarrollo Front-End: Se enfoca en la parte visual y en la interacción del usuario con la página.
    • Desarrollo Back-End: Se encarga de la lógica, el almacenamiento de datos y la seguridad.
    • Full Stack: Combinación de ambos, permitiendo a un programador manejar tanto el diseño como la funcionalidad.
  • ¿Qué es HTML y cómo ha evolucionado hasta HTML5?
    HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y organizar el contenido en la web. HTML ha evolucionado significativamente desde sus inicios en 1991, cuando fue creado por Tim Berners-Lee. Las versiones más importantes incluyen HTML 1.0 (1991), HTML 2.0 (1995), HTML 3.2 (1997), HTML 4.01 (1999) y HTML5 (2014), que introdujo características como soporte para multimedia, etiquetas semánticas y más. HTML5 es la versión más utilizada actualmente y es fundamental para el desarrollo web moderno
  • El rol del navegador web.
    Uno de los roles fundamentales de un navegador es interpretar y renderizar los lenguajes de marcado utilizados en la creación de páginas web, como HTML (HyperText Markup Language), CSS (Cascading Style Sheets) y JavaScript. Este proceso permite al navegador mostrar el contenido de una página web de manera estructurada y visualmente atractiva para el usuario.
  • ¿Qué es la W3C y cuál es su importancia?
    La W3C es fundamental para el desarrollo y crecimiento de la web, ya que establece estándares que aseguran la interoperabilidad y accesibilidad de los sitios web. Estos estándares, como HTML, CSS y XML, son esenciales para que los usuarios puedan acceder a la información de manera rápida y eficiente, sin importar el dispositivo o diseño utilizado por el desarrollador. La W3C también trabaja en la mejora de la seguridad de los sitios web y fomenta la innovación y colaboración entre los miembros de la comunidad web. Su labor es imprescindible para mantener una web abierta, accesible y en constante evolución.
  • La tríada del desarrollo web: HTML, CSS y JavaScript.
    La tríada del desarrollo web se compone de HTML, CSS y JavaScript.

    • HTML (HyperText Markup Language): Estructura y contenido de la página web.
    • CSS (Cascading Style Sheets): Estilo y diseño visual de la página.
    • JavaScript: Interactividad y funcionalidad dinámica en la web.
    Estos tres lenguajes son esenciales para crear una experiencia de usuario completa y atractiva en la web.
  • ¿Qué es y cómo se usa un editor de código (Visual Studio Code)?
    Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Es un editor de código fuente súper rápido y liviano que se puede usar para ver, editar, ejecutar y depurar código fuente para aplicaciones. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.
  • ¿Cómo se usa el inspector del navegador?
    Métodos para abrir el Inspector de Chrome
      Atajo de teclado: Presiona Ctrl + Shift + I (Windows/Linux) o Cmd + Option + I (Mac). Clic derecho: Haz clic derecho en cualquier parte de la página y selecciona «Inspeccionar». Menú del navegador: Ve al menú de opciones (tres puntos en la esquina superior derecha), selecciona «Más herramientas» y luego «Herramientas para desarrolladores».
